图片

说明：

图片的创建方式与超链接相似，和超链接一样，有两种写法，即行内式和参考式写法。
语法中:

  1. Alt：当图片因某些原因不能显示，就用Alt文字来代替图片
  2. Title：鼠标悬停与图片上时出现的文字
  3. Alt 和 Title 都不是必须的，可省略。

操作模块语法
  • start：开始
  • end:结束
  • opration:普通操作块
  • condition:判断块
  • subroutine:子任务块
  • inputoutput:输入输出块
代码：

```flow
st=>start: 开始
e=>end: 结束
op=>operation: 我的操作
cond=>condition: 确认？

st->op->cond
cond(yes)->e
cond(no)->op
```
